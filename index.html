chepherd

/ʃep·hɜrd/ — a TUI supervisor for parallel AI coding agents

A daemon and k9s-style terminal dashboard that watches every Claude Code session you run across all your repos, scores each on goal · velocity · focus · end-state-proximity, and coaches them when they drift — using your own ~/.claude/CLAUDE.md as the rubric.

what it does

─────────────

  • ● watches every Claude Code session in tmux across all your repos — no setup per-session
  • ● reads each session's JSONL transcript + git log + GitHub issue state every tick
  • ● scores G / V / F / E (goal · velocity · focus · end-state-proximity) on every tick
  • ● coaches via tmux paste-buffer when divergence is real — cites principles by number from your CLAUDE.md
  • ● adaptive cadence (2 / 5 / 10 / 30 min) — concentrates attention on struggling sessions, leaves trusted sessions alone
  • ● escalates to Opus when in crisis band, stays on Sonnet otherwise
  • ● anti-theater detection — catches "ship busy commits to deflect from the named blocker" patterns
  • ● never closes issues (only the operator does, after walk-screenshot)

install

─────────

# macOS + Linux — single binary, no runtime deps
curl -fsSL https://chepherd.org/install.sh | sh

# or via Go
go install github.com/chepherd/chepherd@latest

# first-time setup wizard (scans tmux, picks vcs adapter, installs systemd unit)
chepherd init

# open the dashboard
chepherd

under the hood

───────────────

TUI · tview

Same library powering k9s. Mouse + keyboard. True-color when terminal supports it; degrades to 256-color cleanly.

Daemon · Go

Single static binary, ~6 MB. systemd --user / launchd / OpenRC unit. Survives reboots.

Judge · Claude

Subscription-billed via claude-agent-sdk. No API key handling — uses your existing Claude Code credentials.

VCS · pluggable

GitHub (gh), GitLab (glab), Gitea (tea), or .chepherd/issues/*.md when there's no remote at all.

Sessions · adoptable

tmux sessions you already started are auto-discovered. No "use our spawner" lock-in.

Rubric · yours

Your CLAUDE.md is the single source of truth. chepherd cites your principles by number — never paraphrases.

not quite the same as

───────────────────────

Anthropic Agent Teams

In-session lead spawns N teammates inside one task scope. chepherd watches every session you launched yourself, across every repo, with your CLAUDE.md as the rubric — different layer.

Anthropic Routines

Cloud-scheduled fire-and-forget. chepherd is a continuous local watcher.

Overstory / Warren

Spawn their own workers and manage them. chepherd adopts what you already run.
